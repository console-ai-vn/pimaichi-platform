### Funnel Stages

| Stage | Channel | Tactic | Content Type |
|-------|---------|--------|-------------|
| **TOF — Awareness** | Instagram Feed | Regular posting cadence (4-5x/week) | Selfies, lifestyle, fashion (existing formula) |

### Cross-Sell Strategy

- **Post-level:** Every 5th post includes a subtle OF mention in caption
- **Story-level:** Weekly teaser story highlighting exclusive content
- **Highlight-level:** Permanent "Exclusive" highlight with OF previews
- **Live:** Monthly Instagram Live -> tease -> redirect to OF

## 4. Pricing Strategy

### Tiers

```
  $3/mo          $7/mo                     $10/mo
  ┌──────┐      ┌──────────┐             ┌──────────┐
  │ LAUNCH│     │ STANDARD │             │ PREMIUM  │
  │ PROMO │     │ (Anchor) │             │          │
  │ 1st mo│     │ Full     │             │ Standard │
  │ only  │     │ archive  │             │ + DM     │
  └──────┘     │ Stories  │             │ + Custom │
               │ Exclusive│             │   content│
               │ sets      │             │ + Early  │
               └──────────┘             │   access │
                                         └──────────┘
```

### Pricing Psychology

- **$7 anchor:** Feels premium but accessible — positions as "exclusive" not "cheap"
- **$5 option:** For price-sensitive Vietnamese market; frame as "limited rate"
- **$3 promo:** Creates urgency; 67% discount on standard feels like a steal
- **PPV (Pay-Per-View):** $5-$20 per custom set (unlock specific content)
- **Tips:** Encouraged but not required; reward tippers with shoutouts

### Payment Preferences

- Vietnamese users: Prefer bank transfer / VietQR if available via third-party
- International: OnlyFans standard payment (card)
- **Recommendation:** Accept both if possible via OF payment integrations
